# Explainable_Recommender_Systems_-XRS-
Este repositório é um material didático sobre 3 abordagens diferentes de Sistemas de Recomendação Explicáveis (XRS – Explainable Recommender Systems), aplicando cada uma delas sobre o conjunto de dados MovieLens 100k. Faz parte do projeto de extensão da matéria de Sistemas de Recomendação (SCC0284) do ICMC-USP.
